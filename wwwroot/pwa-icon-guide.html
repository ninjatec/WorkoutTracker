How to Create These Icons

You have a few options to create these icon files:

  1. Using an image editor (Photoshop, GIMP, Pixelmator, etc.):
    • Open your logo file (Logo_Without_Words.png)
    • Resize the image to each of the required dimensions
    • Save as PNG with the appropriate filename
  2. Using online PWA icon generators:
    • PWA Asset Generator
    • PWA Builder Image Generator
  3. Using the command line (if you install ImageMagick): 
    brew install imagemagick
    convert Logo_Without_Words.png -resize 144x144 images/icons/icon-144x144.png

Meta Tag Fixes

The console also showed a warning about a deprecated meta tag. To fix it, add this tag to your layout file's head section:

<meta name="mobile-web-app-capable" content="yes">

Testing

After creating these icons and placing them in the correct directories, refresh your application and check if the 404 errors are resolved.
